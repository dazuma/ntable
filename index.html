Ntable

N-dimensional table data structure for Ruby

Download this project as a .zip file Download this project as a tar.gz file

NTable is a Ruby library that provides a convenient data structure for storing n-dimensional tabular data.

It works with zero-dimensional scalar values, arrays, tables, and any arbitrary-dimensional hypertables. Each dimension is described by an axis object. The "rows" in that dimension might be identified by numbers or names. You can perform slice operations across any dimension, as well as reductions and dimensional decomposition. Finally, serialization is provided via a custom JSON schema, as well as a simple "hash of hashes" or "array of arrays" approach.

Install NTable as a gem:

gem install ntable
