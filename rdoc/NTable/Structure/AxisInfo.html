    class NTable::Structure::AxisInfo

    A data structure that provides information about a particular axis/dimension in a Structure. It provides access to the axis object, as well as the axisâ€™s name (if any) and 0-based index into the list of axes. You should never need to create an AxisInfo yourself, but you can obtain one from NTable::Structure#axis.

    Attributes

    axis_index[R]

    The 0-based index of this axis in the structure. i.e. the first, most major axis has number 0.

    axis_name[R]

    The name of this axis in the structure as a string, or nil for no name.

    axis_object[R]

    The underlying axis implementation
